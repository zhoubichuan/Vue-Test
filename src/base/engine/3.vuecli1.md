# 三.View（二）

View 相关属性与作用

## 11.multiWorld

如果为 false，则视图受到约束，因此只能看到一个世界，并且您无法平移边缘
:::demo

```vue {21}
<template>
  <div ref="map" class="map"></div>
</template>
<script>
export default {
  mounted() {
    let {
      Map,
      View,
      layer: { WebGLTile: TileLayer },
      source: { OSM },
    } = ol
    new Map({
      target: this.$refs.map,
      layers: [
        new TileLayer({
          source: new OSM(),
        }),
      ],
      view: new View({
        multiWorld: false,
        center: [12579156, 3274244],
        zoom: 12,
      }),
    })
  },
}
</script>
```

:::

## 12.constrainResolution

如果为真，视图在交互后将始终动画到最近的缩放级别；false 表示允许中间缩放级别
:::demo

```vue {21}
<template>
  <div ref="map" class="map"></div>
</template>
<script>
export default {
  mounted() {
    let {
      Map,
      View,
      layer: { WebGLTile: TileLayer },
      source: { OSM },
    } = ol
    new Map({
      target: this.$refs.map,
      layers: [
        new TileLayer({
          source: new OSM(),
        }),
      ],
      view: new View({
        constrainResolution: true,
        center: [12579156, 3274244],
        zoom: 12,
      }),
    })
  },
}
</script>
```

:::
