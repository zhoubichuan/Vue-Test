# 三.View 属性

View 相关属性与作用

## 1.常用属性

| 属性   | 作用           | 说明                         |
| ------ | -------------- | ---------------------------- |
| center | 视图的初始中心 | -                            |
| zoom   | 缩放级别       | 仅在未定义 resolution 时使用 |
| maxZoom | 用于确定分辨率约束的最大缩放级别 | -    |
| minZoom | 用于确定分辨率约束的最小缩放级别 | -    |


:::demo

```vue {21}
<template>
  <div ref="map" class="map"></div>
</template>
<script>
export default {
  mounted() {
    let {
      Map,
      View,
      layer: { WebGLTile: TileLayer },
      source: { OSM },
    } = ol
    new Map({
      target: this.$refs.map,
      layers: [
        new TileLayer({
          source: new OSM(),
        }),
      ],
      view: new View({
        maxZoom: 14,
        minZoom: 10,
        center: [12579156, 3274244],
        zoom: 12,
      }),
    })
  },
}
</script>
```

:::
