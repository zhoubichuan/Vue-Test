# 二.View(属性一)

View 相关属性与作用

## 1.中心点

| 属性   | 作用           | 配置 |
| ------ | -------------- | ---- |
| center | 视图的初始中心 | -    |

:::demo

```vue {21}
<template>
  <div ref="map" class="map"></div>
</template>
<script>
export default {
  mounted() {
    let {
      Map,
      View,
      layer: { WebGLTile: TileLayer },
      source: { OSM },
    } = ol
    new Map({
      target: this.$refs.map,
      layers: [
        new TileLayer({
          source: new OSM(),
        }),
      ],
      view: new View({
        center: [12579156, 3274244],
        zoom: 12,
      }),
    })
  },
}
</script>
```

:::

## 4.extent

限制视图的程度，换句话说，除此之外，任何东西都在地图上可见
:::demo

```vue {21}
<template>
  <div ref="map" class="map"></div>
</template>
<script>
export default {
  mounted() {
    let {
      Map,
      View,
      layer: { WebGLTile: TileLayer },
      source: { OSM },
    } = ol
    new Map({
      target: this.$refs.map,
      layers: [
        new TileLayer({
          source: new OSM(),
        }),
      ],
      view: new View({
        extent: [-572513.341856, 5211017.966314, 916327.095083, 6636950.728974],
        center: [12579156, 3274244],
        zoom: 12,
      }),
    })
  },
}
</script>
```

:::
