# 图层类别

::: tip 前言
Layer 按照类别，由一个基类加一个或多个子图层类叠加而成

- 瓦片数据 Tile
- 矢量数据 Vector
- 矢量切片 VectorTile
- 图片数据 Image

:::
source 是 Layer 的重要组成部分，表示图层的来源，也就是服务地址
