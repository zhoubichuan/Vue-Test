# 四.Layer（一）

## 1.黑色网格瓦片

黑色网格瓦片是使用 HTML5 画布在客户端上生成的。显示的瓦片坐标是 XYZ 瓦片坐标。
:::demo

```vue
<template>
  <div ref="map" class="map"></div>
</template>

<script>
export default {
  mounted() {
    let {
      Map,
      View,
      layer: { Tile: TileLayer },
      source: { OSM, TileDebug },
    } = ol
    new Map({
      layers: [
        new TileLayer({
          source: new OSM(),
        }),
        new TileLayer({
          source: new TileDebug(),
        }),
      ],
      target: this.$refs.map,
      view: new View({
        center: [12579156, 3274244],
        zoom: 1,
      }),
    })
  },
}
</script>
```

:::
