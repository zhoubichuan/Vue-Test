# 一.TileArcGISRest

- ol.source.TileArcGISRest，ArcGIS Rest 服务提供的切片数据，继承自 ol.source.TileImage

## 1.ArcGIS瓦片图

| ------ | ----- | -------------- | -------------------------------------------------------------------------------------------------------------- |
| layer1 | Tile  | OSM            | -                                                                                                              |
| layer2 | Tile  | TileArcGISRest | "https://sampleserver.arcgisonline.com/ArcGIS/rest/services/" +"Specialty/ESRI_StateCityHighway_USA/MapServer" |
